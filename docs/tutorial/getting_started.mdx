In this introductory tutorial, we will explore the process of configuring the Novorender Web API by developing a plain & simple web application.

While we've only provided tutorials for Vite and Webpack, the setup process should be similar for other bundlers/frameworks, with some minor modifications or adjustments as needed. You can adapt the provided instructions to your specific development environment to seamlessly integrate Novorender into your project.
